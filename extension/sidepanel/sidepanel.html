Axis

Version 2.7.04
Your AI Chrome Agent

Axis lets you control any website hands-free using your voice. Powered by Google Gemini Live API, Axis sees your screen, understands your intent, and takes action — so you never have to touch your keyboard. Axis can now generate stunning images and browse the web with real-time screen understanding.

HOW TO USE
  1. Click Go Live to start a session
  2. Speak your command naturally
  3. Axis will act on the current page
  4. Say “stop” or click End Session when done
POWERED BY
  • Google Gemini Live API
  • Google Cloud Run
  • Chrome Extension MV3
